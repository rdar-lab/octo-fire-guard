## Key Points

1. **Real-time Monitoring**: Plugin receives temperature data every 1-2 seconds
2. **Immediate Response**: Alert and shutdown triggered within milliseconds of threshold breach
3. **Multi-layer Safety**: 
   - Frontend alert (visual + audio)
   - Backend logging
   - Hardware shutdown (GCode or PSU)
4. **User Control**: Fully configurable thresholds and termination methods
5. **Fail-safe**: PSU mode falls back to GCode if PSU control unavailable
